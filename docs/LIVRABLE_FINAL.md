## 🚀 POUR DÉMARRER (5 MIN)

### Étape 1: Créer votre repo GitHub

```bash
git clone <votre-repo-url> MaMutuelle
cd MaMutuelle
```

### Étape 2: Initialiser le projet

```bash
# Si déjà cloné, simplement lire la doc
cat docs/GUIDE_DEMARRAGE_RAPIDE.md
```

### Étape 3: Installer & lancer (30 min)

```bash
# Backend
cd backend
composer install
php artisan key:generate
php artisan migrate

# Frontend
php artisan serve
# Puis ouvrir: http://localhost:8000
```
